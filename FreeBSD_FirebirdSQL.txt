Instalacja
==========

Metoda I
--------
# pkg update
# pkg install firebird25-server

Metoda II
---------
# cd /usr/ports/databases/firebird25-server
# make -DPACKAGE_BUILDING
# make install
# make clean

/etc/rc.conf
------------
firebird_enable="YES"
# opcjonalnie
firebird_mode="superclassic"  #lub "superserver"

# reboot

lub

# /usr/local/etc/rc.d/firebird start
